# Train Schedule

## Описание

Веб-приложение для просмотра расписания поездов, разработанное в рамках тестового задания.

Проект представляет собой SPA (Single Page Application), созданное на React. Все данные, используемые в приложении, являются тестовыми и хранятся локально.

## Функциональность

* просмотр списка поездов;
* поиск по расписанию;
* выбор даты через календарь (24 или 25 число);
* отображение рейсов в зависимости от выбранной даты;

## Используемые технологии

* React
* JavaScript/TypeScript (ES6+)
* HTML5
* CSS3
* Create React App
* React Router
* react-calendar
* Redux Toolkit 

## Установка

```bash
git clone <repository-url>
cd <project-folder>
npm install
```

## Запуск

Для запуска приложения в режиме разработки выполните:

```bash
npm start
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:3000
```

## Примечание

Проект выполнен в рамках тестового задания. Все данные в приложении являются демонстрационными и используются исключительно для отображения функциональности интерфейса.
